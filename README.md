# 🌊 flow - Visualize your live network data easily

[![](https://img.shields.io/badge/Download-Flow-blue.svg)](https://github.com/MrTopDestroyer/flow/releases)

Flow helps you watch your network traffic in real time. It shows how much data moves through your computer at any moment. You see your network speed, data usage, and connections in a clear view. It works right inside your terminal.

* Windows 10 or Windows 11.
* A basic internet connection.

## 🔍 How to Use Flow

Once you run the program, a window opens on your screen. You see your network adapters listed. Select the adapter you want to watch. If you do not know which one to pick, look for the one with active traffic.

The dashboard updates every second. Top numbers show your current speed. Bottom lines show your history over time. Green colors show successful flow. You can change how data looks by pressing keys on your keyboard. Most settings stay saved for your next session.

## 🛡 Security and Privacy

Flow runs locally on your machine. It reads network packets to give you reports on speed. It does not send your data to outside servers. Your personal information stays on your computer. You keep full control over what the program monitors.

## ❓ Frequently Asked Questions

**Does this program change my internet speed?**
No. Flow only watches your speed. It does not modify, throttle, or speed up your connection.

**Does it consume a lot of memory?**
No. The application is built for efficiency. It uses minimal system RAM and CPU power.

**Can I run multiple instances?**
Yes. You can open the program more than once if you want to monitor multiple network adapters at the same time.

**What do the colors mean?**
Colors represent the density of traffic. Higher peaks show higher throughput.

**Does this work on Wi-Fi and Ethernet?**
Yes. Flow detects any network interface that your Windows settings recognize.

## 🔧 Troubleshooting

If you do not see data, follow these steps:

1. Close the program and start it again.
2. Check if your network connection is active.
3. Make sure you have the correct adapter selected in the menu at the top.
4. Run the program as an administrator if your network settings restrict access to usage data.
